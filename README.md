# 🏏 ICC T20 WC Ticket Alert Bot

Monitors BookMyShow for ICC Men's T20 World Cup 2026 semi-final tickets and sends instant Telegram notifications when they go live.

## Features
- Checks every 60 seconds
- Heartbeat every 30 mins
- Sends 3 alerts when tickets are live
- Auto-recovers from errors

## Setup
1. Add your Bot Token and Chat ID in the script
2. Deploy on Railway.app
